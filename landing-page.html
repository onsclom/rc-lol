RC Search

A custom search engine with shortcuts for Recurse Center tools. Add it to your browser for quick access.

Setup

  1. Copy the search URL

    You'll need this URL to configure your browser. It tells the browser where to send your searches.

  2. Add it as a custom search engine

    Pick your browser below and follow the steps.

    Chrome / Brave / Edge

    1. Go to Settings → Search engine → Manage search engines
    2. Click Add under "Site search"
    3. Set the name to RC Search and the shortcut to whatever you like (e.g. rc)
    4. Paste the copied URL into the URL field
    5. Save

    Firefox

    1. Install the Add custom search engine extension
    2. Set the search URL to the copied URL
    3. Give it a name and save

    Safari

    1. Safari doesn't support custom search engines natively
    2. Use an extension like xSearch from the App Store
    3. Configure it with the copied URL

    Arc

    1. Open Settings → Search
    2. Click Add Search Engine
    3. Paste the copied URL and save

  3. Use it

    Type your shortcut keyword (e.g. rc) in the address bar, press Tab or Space, then type your query. Your browser will send it here and you'll be redirected to the right place.
